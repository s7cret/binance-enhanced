Binance strategies workspace

Содержит реализации стратегий, индикаторов, систему бэктестинга и документацию.

Структура:
- strategies/ - реализации стратегий (DCA, Grid, Arbitrage)
- indicators/ - технические индикаторы и шаблоны TradingView (Pine) интеграции
- backtest/ - простой фреймворк для бэктестинга и оптимизации
- reports/ - генерация отчётов по результатам тестов
- docs/ - документация по стратегиям и примеры использования

Примечание: модули написаны так, чтобы быть легко интегрируемыми с вашим пайплайном. Они используют pandas/numpy; при необходимости адаптируйте под свои данные и API.
